# covered

Marketing site for **COVERED** — Free Period Products, Paid for by Advertising.

Single-page static site (`index.html`) with no build step or dependencies.
Live at https://businessforteobaldoselva.github.io/

## View locally

Open `index.html` in any web browser, or serve the folder with any static server.

## Hosting

Published with **GitHub Pages** from the `main` branch. Push to `main` to deploy.

## AI discoverability (AEO/GEO)

The site ships `robots.txt` (AI crawlers allowed), `sitemap.xml`, `llms.txt`
(root and `.well-known/`), an Atom `feed.xml`, and JSON-LD structured data
(Organization, WebSite, FAQPage). Audited at grade A (16/16 foundational
checks) with the `audit-website-aeo` skill from
[gtm-engineer-skills](https://github.com/onvoyage-ai/gtm-engineer-skills).
